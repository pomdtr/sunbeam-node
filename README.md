# Sunbeam Node Sample

## Requirements

- [Sunbeam](https://sunbeam.sh)
- [Node.js](https://nodejs.org)

## Installation

```bash
sunbeam extension install --alias libraries https://raw.githubusercontent.com/pomdtr/sunbeam-node/main/sunbeam.sh
```

## Development

```bash
sunbeam extension install ./extension.js
```
